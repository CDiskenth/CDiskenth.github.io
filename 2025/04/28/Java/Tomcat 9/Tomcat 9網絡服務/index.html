Tomcat中Web應用目錄結構

重點

project/WEB-INF/classes/:Servlet文件存放目錄
project/WEB-INF/lib/:存放運行時需要使用到的jar包
project/WEB-INF/web.xml:Web應用的配置文件

依賴包與API

安裝好Tomcat後，其home目錄下自帶的lib/servlet-api.jar即是依賴包

重點

  • javax.servlet.Servlet:Servlet接口
  • javax.servlet.http.HttpServlet:繼承自javax.servlet.GenericServlet
  • javax.serlvet.RequestDispatcher:轉發器

常用方法:

javax.servlet.Servlet:

Method Arguments Detail
init(config) ServletConfig 創建Servlet對象時調用
service(req,res) ServletRequest,
ServletResponse		 Servlet服務函數，處理數據時使用
destroy() void Servlet對象被銷毀時調用
getServletConfig() void reserved
getServletInfo() void reserved

javax.servlet.http.HttpServletRequest:

Method Arguments Return Detail
getMethod() void String 返回請求方法
getRequestURI() void String 返回URI
getQueryString() void String 返回GET方法的Query參數
getHeader(key) String String 根據key，返回對應請求頭的value
getInputStream() void ServletInputStream 獲取請求體字節輸入流
getReader() void BufferedReader 獲取請求體字符輸入流
getQueryString() void String 獲取網址上之GET請求參數
getServletPath() void String 獲取servlet請求路徑，包含開頭’/‘字符，不含Query參數
getParameter(key) String String 根據請求參數的key獲取對應value
getRequestDispatcher(url) String RequestDispatcher 根據url，獲取轉發器對象
getSession() void HttpSession 從請求對象中獲取Session對象

javax.servlet.http.HttpServletResponse:

Method Arguments Return Detail
serHeader(key,value) void String,String 設置響應頭
getWriter() void PrintWriter 獲取字符輸出流
getOutputStream() void ServletOutputStream 獲取字節輸出流

javax.servlet.RequestDispatcher:

Method Arguments Return Detail
forward(req,res) ServletRequest,ServletResponse void 請求轉發

其他

  • javax.servlet.ServletConfig:每個Servlet的配置信息對象
  • javax.servlet.ServletContext:WEB-INF內的web.xml對象，每個工程中僅有一個此對象
